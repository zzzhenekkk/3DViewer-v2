Detailed Description

Класс для управления настройками визуализации.

Этот класс предоставляет методы для получения и установки различных параметров визуализации, таких как смещение, вращение, масштабирование, цвета и типы линий. Реализует синглтон паттерн для обеспечения единственного экземпляра настроек.
