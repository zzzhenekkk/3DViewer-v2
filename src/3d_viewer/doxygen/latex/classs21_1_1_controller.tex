\doxysubsection{Detailed Description}
Класс контроллера для управления моделью и представлением. 

Этот класс предоставляет функции для загрузки модели, изменения ее положения, вращения, масштабирования, а также получения информации о модели, такой как количество вершин, полигонов и ребер.
